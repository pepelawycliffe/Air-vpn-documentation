How to change the package name?

You can change it easily from Android Studio. Here are the steps:

  • In the Project pane, click on the little gear icon. Uncheck/Deselect the Compact Empty Middle Packages option.
  • Your package directory will now be broken up in individual directories.
  • Individually select each directory you want to rename, and: Right-click itSelect RefactorClick on Rename in the Pop-up dialog.
  • Click on Rename Package instead of Rename Directory Enter the new name and hit Refactor and allow a minute to let Android Studio update all changes.
  • Now open your build.gradle (Usually 'app' or 'mobile'). Update the applicationId to your Package Name and Sync Gradle, if it hasn't already been updated automatically.

Note: When renaming com in Android Studio, it might give a warning. In such case, select Rename All.


How to change app name

  • Open Android Studio > app > res > values > strings.xml
  • Enter your app name inside "app_name" string tag:
    <string name="app_name">Air Vpn</string>

How to change Privacy Policy, Email and app version?

  • Android Studio > app > res > values > strings.xml
  • Enter your app version"app version" string tag:
    <string name="app_version">1.0v</string>
  • Enter your privacy policy url "privacy _policy_url" string tag:
    <string name="privacy_policy_link">Your_Privacy_Policy_Link</string>
  • Enter your email addres "developer_email" string tag:
    <string name="developer_email">Your_Support_Email</string>


How to change Splash Icon, Application Icon and Other icons?

How To Change Application Icon

  • Open Android Studio > app > res > all mipmap folder > ic_launcher.png
  • Rename your square icon with ic_launcher.png and paste in mipmap folder
  • Replace all icons with particular size in particular folder

How To Change Splash Icon

  • Open Android Studio > app > res > drawable > splash_bg.png
  • Rename your splash screen image with splash_bg.png and paste in drawable folder


How to Change AdMob Details?

  • Android Studio > app > res > values > strings.xml
  • Enter your admob app id inside "admob_app_id" string tag:
    <string name="admob_app_id">ca-app-pub-7102591213232959~8245084</string>
  • Enter your admob rewarded video ads id inside "admob_rewarded_video_ads" string tag:
    <string name="admob_rewarded_video_ads">ca-app-pub-3940256099942544/5224354917</string>
  • Enter your admob banner ad id inside "admob_banner_id" string tag:
    <string name="admob_banner_id">ca-app-pub-3940256099942544/6300978111</string>
  • Enter your interstitial ad id inside "admob_interstitial" string tag:
    <string name="admob_interstitial">ca-app-pub-3940256099942544/1033173712</string>
  • Enter your native ad id inside "admob_native_advance" string tag:
    <string name="admob_native_advance">ca-app-pub-3940256099942544/2247696119</string>

How to Generate Signed APK?

Create New Keystore

  • Click Build > Generate Signed APK

  • Select APK and click Next

  • Click On Create New or if you are updating app click choose existing

  • Select Keystore Path
  • Enter Keystore password and confirm password
  • Enter any name for alias
  • Enter key password and confirm password
  • Choose Validity Years
  • Fill rest of the details and click OK

  • Enter keystore password, alias name and key password and click next

  • Choose the signed apk destination
  • Tick the signatures
  • Click Finish

Note :- Please save the keystore and remember the keystore password, alias and key password. If you want to update app in future you have to use this keystore. With new keystore you cannot update your app
